주요 기여

핵심 스테이지 모드 개편

스테이지 수 축소(50→30) 개편에 따른 세이브 데이터 오염 정화·재추첨, 라운드 승패 로직 단일화, 유닛 스케일 조정.

스킨 시스템

영웅·해병 스킨(디바·카우보이·조로 닌자 등)과 스킨 스킬 3종 장착·세이브 통합 마이그레이션, 스킨 전용 스킬 구현.

크래프팅형 이벤트 이식

타 프로젝트의 크래프팅형 라이브 이벤트를 컨베이어 시퀀스·진척도 보상·세이브·IAP 결선까지 포팅.

성능 · 리소스

프로필 아이콘/프레임 스프라이트 로드 배치 병렬화, Resources → Addressable 리소스 이관.

대표 트러블슈팅

테이블 로드 전 이벤트 시즌이 준비돼 무효값이 세이브에 굳던 버그

원인·조치 → 데이터 미로드 상태에서 시즌 준비가 실행돼, 빈 시퀀스의 무효 인덱스(-1)가 세이브에 영속화되며 이벤트가 깨짐. 데이터 미로드·빈 시퀀스 가드를 추가해 무효값 저장을 막고 다음 틱에 재시도하도록 수정.

무기 스킨의 인게임 총 스프라이트가 표시되지 않던 문제

원인·조치 → 스프라이트의 내부 이름(.meta)이 파일명과 달라 이름 기반 인게임 매칭이 실패. 스프라이트 이름을 파일명과 일치시켜 해결.
